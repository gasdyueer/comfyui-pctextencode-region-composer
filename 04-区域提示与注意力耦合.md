# 区域提示与注意力耦合指南

本指南详细说明如何使用 Prompt Control 在提示词中实现区域控制——让画面的不同区域响应不同的提示词。

---

## 两种区域控制方式

PC 提供两种区域提示方法，各有特点：

| 方式 | 语法 | 原理 | 特点 |
|------|------|------|------|
| **Latent 遮罩** | `MASK()` / `AREA()` | 在潜空间上遮罩 | 可与 LoRA 调度结合；区域间可能有轻微渗透 |
| **注意力耦合** | `COUPLE()` | 在注意力层引导 | 通常更精确、更快；可与调度结合 |

---

## 一、MASK 和 AREA：潜空间遮罩/区域

### MASK() — 指定遮罩区域

```
MASK(x1 x2, y1 y2, weight, op)
```

**参数**：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| x1 x2 | 0 1 | 水平范围（左到右） |
| y1 y2 | 0 1 | 垂直范围（上到下） |
| weight | 1 | 遮罩权重 |
| op | multiply | 多遮罩合成操作（multiply/add/subtract/intersect等） |

**值可以是百分比或像素**（不可混用）：
- 百分比：0.0 到 1.0 之间的浮点数
- 像素：大于 1 的整数

> 注意：`1` 被视为百分比而非像素。如需 1 像素，使用绝对像素值但所有值都必须 > 1。

**默认值**：`MASK(0 1, 0 1, 1)` 即全图。可省略尾部参数：`MASK(0 0.5, 0.3)` = `MASK(0 0.5, 0.3 1, 1)`。

### AREA() — 指定计算区域

```
AREA(x1 x2, y1 y2, weight)
```

参数与 MASK 类似，但行为不同。

### MASK 和 AREA 的关键区别

以 1024×1024 的图像为例：

**MASK — 全尺寸计算 + 遮罩叠加**：
```
cat MASK(0 0.5, 0 1) AND dog MASK(0.5 1, 0 1)
```
- 生成两个 1024×1024 的完整输出
- 各自遮罩掉一半后叠加
- 后续步骤仍能看到另一半的内容，可能轻微渗透

**AREA — 分区域独立计算 + 合成**：
```
cat AREA(0 0.5, 0 1) AND dog AREA(0.5 1, 0 1)
```
- 生成两个 512×1024 的独立输出
- 合成到 1024×1024 的潜空间
- 各区域完全独立，不会互相看到

**可以同时使用 MASK 和 AREA**：此时遮罩应用在 AREA 指定的区域内。

### 多遮罩合成

多次使用 MASK/IMASK 时，遮罩会按顺序合成：

```
prompt MASK(0 0.5, 0 1) MASK(0.3 0.7, 0 1, 1, add)
```

合成操作通过 `op` 参数控制，使用 ComfyUI 的 `MaskComposite` 节点操作：
- `multiply`（默认）— 交集
- `add` — 并集
- `subtract` — 差集
- `intersect` — 严格交集
- 等等

多遮罩的整体权重可用 `MASKW(weight)` 设置（默认 1.0）。

## 二、IMASK：使用自定义遮罩

当 MASK 的矩形区域不够用时，可以通过节点传入自定义遮罩。

### 步骤

1. 在工作流中添加 `PC: Attach Mask` 节点，将遮罩连接到 CLIP
2. 在提示词中用 `IMASK(index)` 引用

```
IMASK(index, weight, op)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| index | 0 | 遮罩索引（从 0 开始） |
| weight | 1 | 权重 |
| op | multiply | 合成操作 |

### 示例工作流

```
[Load Image] → [Create Mask] → [PC: Attach Mask] → [PC: Schedule prompt]
                                         ↑
                          (clip 通过此节点传入遮罩)
```

在提示词中：
```
background AND cat IMASK(0)   → cat 只出现在自定义遮罩区域
```

**多遮罩**：使用 `PC: Attach Mask (multi)` 一次附加最多 4 个遮罩，多次使用可附加更多。空输入不计入索引。

## 三、FEATHER：遮罩羽化

```
FEATHER(left top right bottom)
```

对 MASK/IMASK 生成的遮罩应用羽化，使用 ComfyUI 的 `FeatherMask` 节点。值以像素为单位，默认 0。

**与多个遮罩配合**：

FEATHER 按出现顺序对应各遮罩，多余的 FEATHER 应用到合成后的遮罩上：

```
MASK(1) MASK(2) MASK(3) FEATHER(10) FEATHER() FEATHER(5) FEATHER(8)
```
- MASK(1)：羽化 10px
- MASK(2)：FEATHER() 无参数 → 不羽化
- MASK(3)：羽化 5px
- FEATHER(8)：应用到合成后的整体遮罩

FEATHER 和 MASK 的顺序无关，可以交错写。

---

## 四、MASK_SIZE：设置遮罩尺寸

```
MASK_SIZE(width, height)
```

默认遮罩尺寸为 512×512，ComfyUI 会自动缩放到实际图像分辨率。

- 可通过 `PC: Configure PCTextEncode` 节点修改默认值
- 也可在提示词中使用 `MASK_SIZE()` 临时覆盖

## 五、COUPLE：注意力耦合（实验性）

> ⚠️ 实验性功能，语法可能变更。

注意力耦合是一种基于注意力机制的区域提示实现，通常比潜空间遮罩更快、更精确。

### 基本语法

```
基础提示词 COUPLE 耦合提示词1 COUPLE 耦合提示词2
```

**带遮罩参数的快捷写法**：
```
基础提示词 COUPLE(0.5 1) 上半部分内容 COUPLE(0 0.5) 下半部分内容
```
`COUPLE(maskparams)` 等同于 `COUPLE MASK(maskparams)`。

### 示例

```
landscape COUPLE(0.5 1) a cat sitting on a tree COUPLE(0 0.5) a dog running on grass
```
- 基础提示 "landscape" 作用于全图
- "cat" 被注意力耦合到上半部分
- "dog" 被注意力耦合到下半部分

### 行为要点

- **未指定遮罩**：如果 COUPLE 后没有 MASK，会隐式使用 `MASK()`
- **FILL()**：基础提示词使用 `FILL()` 可自动填充未被耦合提示遮罩覆盖的区域
- **权重为 0**：如果基础提示词权重为 0（`:0`），第一个非零权重的耦合提示词会成为基础提示

```
dog FILL() COUPLE(0.5 1) cat
```
→ dog 填充下半部分，cat 填充上半部分

### 配合自定义遮罩

```
base prompt COUPLE IMASK(0) region 1 content COUPLE IMASK(1) region 2 content
```

### 批量负面条件

使用 `PC: Attention Couple (batch negative)` 节点可以将注意力耦合应用到负面条件，效果与原版 pamparamm 实现一致且性能更好：

```
[PC: Schedule prompt] → positive ──┐
                                    ├── [PC: Attention Couple (batch negative)] → positive, negative
[CLIPTextEncode]      → negative ──┘
```

如果无法批量处理，此节点会安全回退到默认行为。

### 注意事项

- 注意力耦合不保证完美限制提示词效果到遮罩区域，因为生成过程仍然看到完整潜空间
- COUPLE 可用于负面提示词，会正确工作
- 与 LoRA 调度完全兼容

## 六、综合示例

### 示例 1：简单的左右分割

```
a forest AND a cat MASK(0 0.5, 0 1) AND a dog MASK(0.5 1, 0 1)
```
左半边是猫，右半边是狗，背景都是森林。

### 示例 2：注意力耦合 — 角色与背景分离

```
beautiful landscape FILL() COUPLE(0.2 0.8, 0.3 0.9) a girl in red dress
```
背景填充整个画面，女孩只出现在中间区域。

### 示例 3：带调度的区域提示

```
[background AND cat MASK(0 0.5, 0 1) AND dog MASK(0.5 1, 0 1):background AND only a cat MASK(0 1, 0 1):0.5]
```
前 50% 有猫和狗分别在左右两侧，后 50% 只有猫占满全图。

### 示例 4：自定义遮罩 + 羽化

```
background AND character IMASK(0) FEATHER(20 20 20 20)
```
角色出现在自定义遮罩区域，边缘羽化 20 像素以平滑过渡。

### 示例 5：多区域 COUPLE

```
garden COUPLE(0 0.33, 0 1) red roses COUPLE(0.33 0.66, 0 1) white lilies COUPLE(0.66 1, 0 1) blue violets
```
花园作为基础，三种花分别占据左、中、右三个区域。
