# ComfyUI Prompt Control 快速入门

## 这是什么？

ComfyUI Prompt Control（简称 PC）是一个强大的 ComfyUI 扩展，让你**仅通过提示词文本**就能控制 LoRA 加载调度、提示词切换、区域提示、注意力耦合等功能——不用再连一堆节点线！

简单说：**一段提示词 = 一张复杂的节点连线图**。

## 最低要求

- ComfyUI v0.8.0 或更高版本

## 核心理念

PC 提供了两类节点：

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| **Lazy 节点**（调度节点） | 支持提示词调度、LoRA调度、宏展开 | 日常使用首选 |
| **PCTextEncode**（编码节点） | 只编码单段提示词，不支持调度语法 | 作为 Lazy 节点的底层，或只需要高级编码功能时 |

> **新手建议**：直接使用 Lazy 节点即可，它们覆盖了所有功能。

## 最简单的用法：替换 CLIPTextEncode

把工作流中的 `CLIPTextEncode` 替换为 `PC: Schedule prompt`（即 `PCLazyTextEncode`），输入和输出完全一样——但你现在拥有了所有高级功能。

```
最简单的提示词（无调度，等同于 CLIPTextEncode）：
a beautiful cat sitting on a windowsill
```

## 第一个调度：提示词切换

```
a [red:blue:0.5] cat
```

这表示：采样前 50% 用 "a red cat"，后 50% 切换为 "a blue cat"。

## 第一个 LoRA 调度

```
a cat <lora:catlora:1> [<lora:detailer:1>::0.5]
```

- `catlora` 全程生效
- `detailer` 只在前 50% 生效

要使用 LoRA 调度，需要搭配 `PC: Schedule LoRAs`（即 `PCLazyLoraLoader`）节点。

## 完整的最小工作流

```
┌─────────────┐     ┌──────────────────────┐     ┌───────────┐
│ Checkpoint  │────►│ PC: Schedule LoRAs    │────►│ KSampler  │
│ Loader      │     │ (model + clip 输出)    │     │           │
└─────────────┘     └──────────────────────┘     └─────┬─────┘
                                                     │
┌─────────────┐     ┌──────────────────────┐          │
│ (同上 clip)  │────►│ PC: Schedule prompt   │──────────┘
│             │     │ (conditioning 输出)    │
└─────────────┘     └──────────────────────┘
```

1. **Checkpoint Loader** 输出 model 和 clip
2. **PC: Schedule LoRAs** 接收 model、clip 和含 LoRA 标记的提示词，输出带 LoRA 的 model 和 clip
3. **PC: Schedule prompt** 接收 clip 和提示词，输出 conditioning
4. 将以上输出接入 KSampler 即可

## 5 分钟速查：语法一览

| 语法 | 作用 | 示例 |
|------|------|------|
| `[before:after:X]` | 在 X 处切换提示词 | `[cat:dog:0.5]` |
| `[a\|b:X]` | 交替提示词 | `[red\|blue:0.1]` |
| `<lora:名称:权重>` | 加载 LoRA | `<lora:detail:0.8>` |
| `[<lora:x:1>::0.5]` | 调度 LoRA | 前50%启用 |
| `AND` | 组合多段提示词 | `cat AND dog` |
| `BREAK` | 分块编码 | `cat BREAK dog` |
| `(word:1.5)` | 权重强调 | `(red:1.3)` |
| `STYLE(A1111)` | 改变权重解析方式 | `STYLE(A1111)` |
| `DEF(X=...)` | 定义宏 | `DEF(CAT=a cute cat)` |
| `MASK(...)` | 区域遮罩 | `MASK(0 0.5, 0 1)` |
| `COUPLE(...)` | 注意力耦合 | `COUPLE(0.5 1)` |
